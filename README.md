CRICAUCTIONIPL
CRICAUCTIONIPL is a web-based application and game that simulates an IPL-style cricket auction using real auction rules.
Users can participate in mock auctions, compete against AI, create or join auction rooms, and manage teams in a realistic IPL environment.

How to Run the Project Locally
Make sure Node.js and npm are installed.

# Clone the repository
git clone <YOUR_GIT_URL>

# Navigate to the project folder
cd CRICAUCTIONIPL

# Install dependencies
npm install

# Start the development server
npm run dev"# CricAuctionIPL"
